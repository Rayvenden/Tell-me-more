Demo of Tell-me-more

python3 main.py --bash displays the bash users of the system.

python3 main.py --bash
root
Trishna Guha

python3 main.py --whoam works as "whoami" command.
This displays the current user who is logged in the system

python3 main.py --whoam
trishna

python3 main.py --mem provides the information about total memory, available memory and free memory of the system.

python3 main.py --mem
MemTotal: 3846 MB
MemFree: 2207 MB
MemAvailable: 2874 MB
