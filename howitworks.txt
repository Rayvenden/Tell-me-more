Demo of Tell-me-more

python3 main.py --users displays the bash users of the system.

python3 main.py --users / python3 main.py -u
root
Trishna Guha

python3 main.py --nbusers displays the users who are not using bash, excluding the accounts with the nologin shell.

python3 main.py --nbusers / python3 main.py -n
sync
shutdown
halt

python3 main.py --whoami works as "whoami" command.
This displays the current user who is logged in the system

python3 main.py --whoami / python3 main.py -i
trishna

python3 main.py --mem provides the information about total memory, available memory and free memory of the system.

python3 main.py --mem / python3 main.py -m
MemTotal: 3846 MB
MemFree: 2207 MB
MemAvailable: 2874 MB
